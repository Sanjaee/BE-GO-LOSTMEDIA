# Pkg - Public Packages

Reusable packages that can be used across different projects.

## Structure

- `database/` - Database connection, migrations, and query builders
- `cache/` - Redis cache wrapper and utilities
- `mq/` - RabbitMQ publisher and consumer utilities
- `storage/` - Cloudinary integration for file storage
- `search/` - Meilisearch integration for search functionality
- `logger/` - Zap logger setup and utilities
